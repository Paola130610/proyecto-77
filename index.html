Escribe sobre visual studio
visual studio es un IDE que nos permite escribir código sin la necesidad de internet
Pasos para configurarlo:
1. instalar el IDE visual studio code.
2. abrir el IDE.
3. agregar live server desde exenciones.
4. abrir la carpeta donde vamos a guardar nuestros archivos.
5. agregar los archivos con su extensipon.
6. probar nuestro html vale usando el botón de go live.
